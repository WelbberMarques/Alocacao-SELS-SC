Sistema de preenchimento automatico de planilhas de alocacao via scraping do GSA.

## Tecnologias

- Python + Tkinter/CustomTkinter
- Selenium (automacao GSA)
- Supabase (autenticacao e banco de dados)
- openpyxl (manipulacao de planilhas)
- PyInstaller (geracao de executavel)

## Funcionalidades

- Login com autenticacao segura (bloqueio por IP, VPN detection)
- Processamento paralelo de multiplos XMLs
- Preenchimento automatico de planilhas de alocacao
- Painel administrativo (Master)
- Auto-update via Supabase Storage
